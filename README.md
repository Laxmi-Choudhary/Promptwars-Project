🚀 CrowdSense AI – Smart Crowd Anomaly Detection System
📌 Overview
CrowdSense AI is an intelligent, AI-powered system designed to monitor and analyze crowd behavior in real time during large-scale events such as stadiums, concerts, and festivals. The system detects unusual patterns (anomalies) in crowd movement and helps event organizers take proactive actions to ensure safety, reduce congestion, and improve the overall attendee experience.

🎯 Problem Statement
Managing large crowds at physical events is challenging due to:

Overcrowding at entry/exit points

Long waiting times

Lack of real-time monitoring

Delayed response to emergency situations

This project addresses these issues by providing real-time crowd analysis and anomaly detection.

💡 Solution
CrowdSense AI uses Machine Learning to continuously monitor crowd data across different zones and identify abnormal patterns. It provides instant alerts, enabling quick decision-making and better crowd control.

⚙️ Features
📊 Real-time crowd monitoring

🧠 AI-based anomaly detection using Isolation Forest

📍 Multi-zone tracking (e.g., Gate A, Food Court, Exit)

🚨 Instant alert system for unusual activity

📈 Interactive dashboard using Streamlit and Plotly

⏱️ Time-series data analysis
